  MUSE V3 BAND APP - МОБИЛЬНОЕ ПРИЛОЖЕНИЕ ДЛЯ МОНИТОРИНГА БРАСЛЕТА
═══════════════════════════════════════════════════════════════════════════

НАЗНАЧЕНИЕ:
Приложение разработано для управления и мониторинга умного браслета Muse v3 
через Bluetooth Low Energy (BLE), предназначенного для отслеживания детей 
в возрасте 1-12 лет.

ОСНОВНОЙ ФУНКЦИОНАЛ:

1. СКАНИРОВАНИЕ И ПОДКЛЮЧЕНИЕ
   Приложение сканирует BLE устройства в радиусе действия и позволяет 
   подключиться к браслету Muse v3 одним нажатием.

2. МОНИТОРИНГ СОСТОЯНИЯ УСТРОЙСТВА
   Отображает ключевые параметры: уровень батареи, версию прошивки 
   (bootloader, application, Bluetooth stack) и текущее состояние системы.

3. СОХРАНЕНИЕ ПОСЛЕДНИХ ДАННЫХ
   Реализован механизм сохранения последних известных значений для каждого 
   параметра - даже если новый запрос не вернул данные, отображаются 
   предыдущие значения с меткой времени последнего обновления.

4. АВТОМАТИЧЕСКОЕ ОБНОВЛЕНИЕ
   Встроенная функция автообновления позволяет запускать периодический опрос 
   устройства каждые 5 секунд для мониторинга батареи в реальном времени.

5. REAL-TIME STREAMING СЕНСОРОВ
   Приложение поддерживает потоковую передачу данных со всех доступных 
   сенсоров браслета: акселерометр, гироскоп, магнетометр, датчики 
   температуры, влажности, давления и освещенности.

6. РЕЖИМЫ STREAMING
   Поддерживаются два режима: Buffered Mode (буферизованная передача с 
   меньшей нагрузкой на BLE) и Direct Mode (прямая передача в реальном 
   времени с минимальной задержкой).

7. ДАТЧИКИ ДВИЖЕНИЯ
   Отображение данных гироскопа (угловая скорость в dps), акселерометра 
   (ускорение в mg) и магнетометра (магнитное поле в mGauss) с визуализацией 
   по осям X/Y/Z.

8. ЭКОЛОГИЧЕСКИЕ ДАТЧИКИ
   Мониторинг температуры (°C), влажности (%), атмосферного давления (hPa), 
   освещенности (lux) и расстояния (mm) до объектов.

9. КАЧЕСТВО ВОЗДУХА (AQI EXPANSION)
   При наличии расширения AQI показываются данные CO₂, VOC (летучие 
   органические соединения), пыли PM1/PM2.5/PM10 и угарного газа CO.

10. ВИЗУАЛИЗАЦИЯ ДАННЫХ
    Для акселерометра реализованы мини-графики истории последних 50 семплов 
    с отдельными кривыми для каждой оси, что позволяет отслеживать паттерны 
    движения.

11. ГИБКАЯ НАСТРОЙКА STREAMING
    Экран конфигурации позволяет выбрать любую комбинацию сенсоров, частоту 
    сбора данных (25-1600 Hz) и режим передачи с автоматической валидацией 
    размера пакета.

12. ИНДИКАТОРЫ ЗАГРУЗКИ
    Каждый параметр имеет индивидуальный индикатор загрузки, показывающий 
    процесс обновления данных в реальном времени.

13. МЕТКИ ВРЕМЕНИ
    Все данные отображаются с меткой последнего обновления в удобном формате 
    ("5s ago", "2m ago" или точное время).

14. СТАТИСТИКА STREAMING
    Во время streaming отображается количество полученных пакетов, длительность 
    сеанса и статус подключения.

15. ПРОТОКОЛ MUSE V3
    Полная реализация коммуникационного протокола Muse v3 на основе TLV 
    (Type-Length-Value) формата с поддержкой всех команд чтения и управления.

16. УПРАВЛЕНИЕ СОСТОЯНИЯМИ
    Приложение корректно работает со всеми состояниями устройства: IDLE, 
    STANDBY, LOG, TX (BUFFERED/DIRECT), READOUT и CALIBRATION.

17. ДЕТЕКЦИЯ ПАДЕНИЯ (MAN DOWN)
    Поддержка алгоритма детекции падения с отображением уровня тревоги 
    (0-3) и статуса готовности алгоритма.

18. ОРИЕНТАЦИЯ В ПРОСТРАНСТВЕ
    При наличии MPE firmware отображается ориентация браслета в виде 
    кватерниона (W, I, J, K) для точного определения положения.

19. СОВРЕМЕННЫЙ UI/UX
    Интерфейс выполнен в стиле Material Design 3 с цветовой индикацией, 
    прогресс-барами, градиентами и интуитивной навигацией.

20. НАДЕЖНОСТЬ И СТАБИЛЬНОСТЬ
    Приложение корректно обрабатывает потерю BLE пакетов, тайм-ауты команд, 
    отключение устройства и восстанавливает данные при возобновлении связи.

ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ:
- Платформа: Flutter (iOS/Android)
- BLE Protocol: Muse v3 Communication Protocol v2.24
- Частота обновления: до 1600 Hz
- Количество сенсоров: 13+ типов данных
- Размеры пакетов: 6/12/24/30/60 байт
